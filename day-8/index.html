Definitions

Constructor Function

A constructor function is a special function used as a blueprint to create multiple objects with similar properties and behavior.

Prototype

A prototype is an object attached to a constructor that allows all created objects to share common methods and properties, saving memory.

IMP Questions & Answers


Q1. Why do we use constructor functions?
: We use constructor functions to create multiple objects with the same structure and properties without repeating code.
Q2. What is the purpose of the new keyword?
: The new keyword creates a new object, binds this to that object, executes the constructor function, and returns the newly created object.
Q3. Why do we use prototypes?
: We use prototypes to share methods and properties among all objects created from the same constructor, which improves memory efficiency.
Q4. Where does JavaScript look when a property is not found in an object?
: When a property is not found in an object, JavaScript searches for it in the object's prototype and continues up the prototype chain if necessary.
Q5. What is a prototype chain?
: A prototype chain is the mechanism by which JavaScript searches for properties and methods by moving from an object to its prototype and then to higher-level prototypes until the property is found or the chain ends.
What is a Prototype Chain? : A prototype chain is the sequence of objects that JavaScript follows to search for a property or method when it is not found directly on an object.
